# Agent Instructions

## Critical Workflow Rules

### WRITEUP.md Protocol
**NEVER write directly to WRITEUP.md without explicit user approval.**

All text must be proposed in the chat conversation first. The user must explicitly approve (e.g., "cool", "approve", "looks good", "write it") before any content is written to WRITEUP.md.

This applies to all sections: Abstract, Why I Built This, Background, Solution, Technical Details, Experiments, and Future Directions.

Violating this rule damages trust and wastes the user's time.
